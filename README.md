# Calendar
A nice solar and lunar calendar application with React Native


## State management:
Redux

## Install
```
npm install
cd ios
pod install
```

## Build
1. Open XCode and signing with your apple id
2. Connect your device via Type-C port
3. Build Shortcut: command + B
